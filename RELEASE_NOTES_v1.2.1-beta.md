# Pool Pilot v1.2.1-beta

## ✨ New features
- New `select` entity to choose between centered automatic filtration and a bounded time window.
- New `time` entities for the minimum start time and maximum end time.
- The three scheduling settings can now be controlled directly from the Pool Pilot Dashboard.

## 🐞 Bug fixes
- Fixed the missing Dashboard control for the automatic-filtration placement mode and its minimum/maximum times.
- Existing v1.2.0-beta.1 settings are preserved automatically.
