# Pool Pilot v1.2.0-beta.1

Pre-release for community testing.

## New features
- User-selected ORP or free-chlorine measurement mode.
- Simple and advanced electrolyzer profiles.
- Advanced electrolyzer entities: production percentage, Boost and status.
- Automatic filtration can remain centered or be constrained to a start/end window.

## Compatibility
Existing v1.1.x configurations are kept compatible. Open the integration options once to review the new settings.
